# Qwen Code - Проект mh-compressor-manager

## О проекте
Автоматическое сжатие статического контента для веб-сервера nginx с целью снижения сетевого трафика и ускорения загрузки страниц.

**Язык:** C++23
**Сборка:** CMake
**Платформа:** Linux (Fedora/RHEL)

## Структура правил
Правила разделены по файлам для удобства:

| Файл | Назначение |
|------|-----------|
| **QWEN.md** | Общая информация о проекте и навигация |
| **TECHNICAL_SPECIFICATION.md** | Полная техническая спецификация проекта (требования, архитектура, критерии приемки) |
| **RULES.md** | Правила кода: стиль, именование, комментарии |
| **DEPLOY.md** | Сборка, деплой, SSH команды |
| **CONTRIBUTING.md** | Pull Request, коммиты, автоматизация |

## Быстрый старт

Подробные инструкции см. в соответствующих файлах:
- **Техническая спецификация:** см. `TECHNICAL_SPECIFICATION.md`
- **Сборка и деплой:** см. `DEPLOY.md`
