# Qwen Code - Проект mh-compressor-manager

## О проекте
Автоматическое сжатие статического контента для веб-сервера nginx с целью снижения сетевого трафика и ускорения загрузки страниц.

**Язык:** C++23
**Сборка:** CMake
**Платформа:** Linux (Fedora/RHEL)

## Цикл аудита
Процедура аудита определена в [audit/AUDIT_CYCLE.md](../../audit/AUDIT_CYCLE.md).

**Два режима:**
1. **Стандартный**: аудит → исправление одной ошибки → пуш → сброс → повтор
2. **С полным тестированием**: аудит → тесты → исправление → пуш → сброс → повтор

**Правила:**
- Одно исправление за раунд (не массовые изменения)
- Сброс контекста между раундами
- Журналирование в audit/CHANGELOG.md
- Подсчёт раундов в audit/ROUND_HISTORY.md
