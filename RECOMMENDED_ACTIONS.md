# Список рекомендуемых действий

Отправьте **`?`** в чат, чтобы AI-ассистент снова показал этот список.

```
🔄 1) Проведи цикл аудита
🔍 2) Проведи полный аудит
🧪 3) Проведи цикл аудита с полным тестированием
🚀 4) Проведи нагрузочное тестирование (1GB+)
🛡️ 5) Проведи аудит безопасности (CWE/OWASP, penetration test)
📈 6) Проведи бенчмарк производительности (сравнение до/после оптимизаций)
📝 7) Подготовь отчёт для Changelog и Release Notes
📦 8) Собери RPM-пакет и протестируй установку
🧹 9) Очисти мёртвый код и неиспользуемые зависимости
📊 10) Контекстная альтернатива (AI предложит на основе ситуации)
```

> **Примечание:** Пункты 1-9 — стандартные действия.
> На основе контекста и предыдущих действий AI-ассистент может предложить альтернативные варианты.

### Описание действий

| № | Действие | Описание |
|---|----------|----------|
| 1 | **Цикл аудита** | Быстрая проверка по AUDIT_CYCLE.md — 10-20 мин |
| 2 | **Полный аудит** | Глубокая проверка по AUDIT_FULL_CYCLE.md — 30-60 мин |
| 3 | **Цикл аудита с тестированием** | Аудит + запуск всех тестовых скриптов |
| 4 | **Нагрузочное тестирование** | 1GB+ данных, замер производительности и стабильности |
| 5 | **Аудит безопасности** | CWE/OWASP checklist, penetration test, поиск уязвимостей |
| 6 | **Бенчмарк** | Сравнение производительности до/после оптимизаций |
| 7 | **Changelog и Release Notes** | Подготовка отчёта для релиза |
| 8 | **RPM-пакет** | Сборка и тестирование установки пакета |
| 9 | **Очистка мёртвого кода** | Удаление неиспользуемых функций, include, зависимостей |
| 10 | **Контекстная альтернатива** | AI предлагает вариант на основе текущей ситуации и истории действий |
